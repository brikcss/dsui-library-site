Typography

UX Guidelines

  • DS UI closely follows typography specs for MD.
  • DS UI guidelines take precedence, but MD's typography specs are considered an extension of DS UI.
  • Roboto is the only font family that should be used. Any exceptions require UX approval.
  • Do not write application-specific CSS for typography styles. Instead, use the provided HTML classes and SASS helpers, which provide all UX-approved styles for typography. Any exceptions require UX approval.
  • Be sure to follow any listed use case guidelines when applying typography styles.

Install

It is strongly encouraged to install this with DS UI Core:

npm install @directscale/core --save-dev

You may also install it standalone:

npm install @directscale/typography --save-dev

Basic Usage - HTML Classes

Caption font style

When to use? Captions. Legal text. Use sparingly.

<p class="font__caption">Caption: Regular 12sp</p>

Button font style

When to use? Buttons.

<p class="font__button">Button: MEDIUM (ALL CAPS) 14sp</p>

Body font style

When to use? This is the font style applied to the body tag by default.

<p class="font__body">Body: Regular 14sp</p>

Bold font style

When to use? Bold text.

<p class="font__bold">Bold: Medium 14sp</p>

Subheading font style

When to use? No use cases specified.

<p class="font__subheading">Subheading: Regular 16sp</p>

Title font style

When to use? App bar (header) title.

<p class="font__title">Title: Medium 20sp</p>

Headline font style

When to use? No use cases specified.

<p class="font__headline">Headline: Regular 24sp</p>

Display 1 font style

When to use? No use cases specified.

<p class="font__display1">Display 1: Regular 34sp</p>

Display 2 font style

When to use? No use cases specified.

<p class="font__display2">Display 2: Regular 45sp</p>

Display 3 font style

When to use? No use cases specified.

<p class="font__display3">Display 3: Regular 56sp</p>

Display 4 font style

When to use? No use cases specified.

<p class="font__display4">Display 4: Light 112sp</p>

Basic Usage - SASS Helpers

In addition to the HTML classes above, SASS helpers are available for more control.

sp($name) function

This function applies the correct font-size for a given font style to any element. For example:

// Will return a font-size of `16sp` in `em` units.
.my-custom-selector {
	font-size: sp('subheading');
}

@include font($style) mixins

This mixin applies a font style to a selector.

.my-custom-selector {
	@include font('display');
}
.my-custom-selector {
	/* This is equal to 34px/34sp. */
	font-size: 2.42857em;
	line-height: 1.5rem;
	color: hsla(0, 0%, 0%, 0.54));
}
