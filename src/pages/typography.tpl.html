Guidelines

Font Face

Roboto is the only font family that should be used. Any exceptions require UX approval.

Font Styles

Font styles should only be selected from the UX approved list of predefined styles, outlined on this page. In other words, do not write application-specific CSS for typography styles. Instead, use the provided classes and helpers. Any exceptions require UX approval.

Follow any listed use case guidelines when applying typography styles.

Text Colors

Text colors are part of the Colors component.

Line Length / Readability

For best readability, lines of text should be no longer than ~60 characters. Shorter lines of text should wrap at ~30 characters or less.

Additional Guidelines

MD typography specs are an extension of DS UI, and should be referred to for further guidance.

Install

It is strongly encouraged to install typography with DS UI Core:

npm install @directscale/core --save-dev

You may also install typography on its own:

npm install @directscale/typography --save-dev

Setup

  1. Copy typography/*.{woff,woff2} files to your <build>/css/fonts folder. This can be automated for you be putting it into your build.
  2. Include _typography.abstract.scss in your SASS build.
  3. Initialize typography styles by doing one of the following:
    • Basic: Include _typography.init.scss into your SASS build.
    • Advanced: Use _typography.init.scss as an example to initialize typography styles on your own.

Basic Usage - HTML Classes

Font Style Classes

.font__caption Caption font style

When to use? Captions. Legal text. Use sparingly.

<p class="font__caption">Caption: Regular 12sp</p>

.font__button Button font style

When to use? Buttons. Tabs.

<p class="font__button">Button: MEDIUM (ALL CAPS) 14sp</p>

.font__body Body font style

When to use? This is the font style applied to the body tag by default.

<p class="font__body">Body: Regular 14sp</p>

.font__bold Bold font style

When to use? Bold text. Text with emphasis. Ask UX for specific use cases.

<p class="font__bold">Bold: Medium 14sp</p>

.font__subheading Subheading font style

When to use? Subheadings, such as widget section subheadings (see Marketing Sites widget, Community widget).

<p class="font__subheading">Subheading: Regular 16sp</p>

.font__title Title font style

When to use? App bar (header) title. Widget title when not in a header (see Social Networking widget). Rank titles (see rank titles in Rank Information widget).

<p class="font__title">Title: Medium 20sp</p>

.font__headline Headline font style

When to use? Widget header title.

<p class="font__headline">Headline: Regular 24sp</p>

.font__display1 Display 1 font style

When to use? Use sparingly, with caution, when needing to call out large text.

<p class="font__display1">Display 1: Regular 34sp</p>

.font__display2 Display 2 font style

When to use? Avoid use. Get UX approval for use cases.

<p class="font__display2">Display 2: Regular 45sp</p>

.font__display3 Display 3 font style

When to use? Avoid use. Get UX approval for use cases.

<p class="font__display3">Display 3: Regular 56sp</p>

.font__display4 Display 4 font style

When to use? Avoid use. Get UX approval for use cases.

<p class="font__display4">Display 4: Light 112sp</p>

Text body length classes

.font__short-line

When to use? For short lines of text, to restrict its length to ~30 characters.

<p class="font__short-line"><em>When to use?</em> For short lines of text (like this line), to restrict its length to ~30 characters.</p>

Basic Usage - SASS Helpers

In addition to the HTML classes above, SASS helpers are available for more control.

sp($name) function

This function applies a font-size for a predefined font style to any element. This function should replace using font units (px, etc.) anywhere in your app code. For example:

.my-custom-selector {
			font-size: sp('subheading');
		}
.my-custom-selector {
			/* Assuming a base font size of 14px, the compiled result will be: */
			font-size: 1.14286em;
		}

@include font($style) mixin

This mixin applies a predefined font style to a selector.

.my-custom-selector {
			@include font('display');
		}
.my-custom-selector {
			/* Assuming a base font size of 14px, the compiled result will be: */
			font-size: 2.42857em;
			line-height: 1.5rem;
			color: hsla(0, 0%, 0%, 0.54));
		}
